# Практическая работа. Медиа запросы в CSS

## Тема
Адаптивная верстка с использованием медиа запросов CSS

## Цель
Освоить принципы создания адаптивных веб-страниц с помощью медиа запросов, научиться создавать интерфейсы, которые корректно отображаются на различных устройствах и размерах экранов.

## Теоретическая часть

### Что такое медиа запросы?
Медиа запросы (Media Queries) - это модуль CSS3, который позволяет адаптировать содержимое веб-страницы к различным характеристикам устройств, таким как ширина экрана, высота, ориентация и разрешение.

### Синтаксис медиа запросов
```css
@media [тип носителя] and (условие) {
  /* CSS-правила */
}
```

### Основные типы условий:
- `min-width` - минимальная ширина окна
- `max-width` - максимальная ширина окна
- `orientation` - ориентация (portrait/landscape)
- `aspect-ratio` - соотношение сторон

### Типичные breakpoints:
```css
/* Мобильные устройства */
@media (max-width: 767px) { ... }

/* Планшеты */
@media (min-width: 768px) and (max-width: 1023px) { ... }

/* Десктопы */
@media (min-width: 1024px) { ... }
```

## Практический пример

### Базовый HTML:
```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Адаптивная страница</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header class="header">
        <h1>Мой адаптивный сайт</h1>
        <nav class="navigation">
            <a href="#">Главная</a>
            <a href="#">О нас</a>
            <a href="#">Контакты</a>
        </nav>
    </header>
    <main class="main-content">
        <section class="content">
            <h2>Основной контент</h2>
            <p>Адаптируемый текст...</p>
        </section>
        <aside class="sidebar">
            <h3>Боковая панель</h3>
            <p>Дополнительная информация...</p>
        </aside>
    </main>
    <footer class="footer">
        <p>&copy; 2024 Все права защищены</p>
    </footer>
</body>
</html>
```

### Базовый CSS с медиа запросами:
```css
/* Базовые стили для мобильных устройств */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

.header {
    background: #333;
    color: white;
    padding: 1rem;
    text-align: center;
}

.navigation {
    display: flex;
    flex-direction: column;
    gap: 0.5rem;
    margin-top: 1rem;
}

.navigation a {
    color: white;
    text-decoration: none;
    padding: 0.5rem;
    background: #555;
}

.main-content {
    padding: 1rem;
}

.content, .sidebar {
    padding: 1rem;
    margin-bottom: 1rem;
}

.content {
    background: #f4f4f4;
}

.sidebar {
    background: #e4e4e4;
}

.footer {
    background: #333;
    color: white;
    text-align: center;
    padding: 1rem;
}

/* Планшеты */
@media (min-width: 768px) {
    .navigation {
        flex-direction: row;
        justify-content: center;
    }
    
    .main-content {
        display: flex;
        gap: 2rem;
    }
    
    .content {
        flex: 2;
    }
    
    .sidebar {
        flex: 1;
    }
}

/* Десктопы */
@media (min-width: 1024px) {
    .header {
        display: flex;
        justify-content: space-between;
        align-items: center;
    }
    
    .navigation {
        margin-top: 0;
    }
    
    .container {
        max-width: 1200px;
        margin: 0 auto;
    }
}
```

## Ход работы

1. Создайте базовую HTML-структуру
2. Добавьте базовые CSS-стили (mobile-first)
3. Реализуйте медиа запросы для планшетов

5. Протестируйте на различных размерах экрана

## Критерии оценки

**5 (Отлично):**
- Полностью работоспособный адаптивный интерфейс
- Корректное отображение на 3+ размерах экрана
- Чистый, хорошо структурированный код
- Правильное использование медиа запросов
- Наличие всех требуемых файлов в репозитории

**4 (Хорошо):**
- Работоспособный адаптивный интерфейс
- Корректное отображение на 2+ размерах экрана
- Незначительные недочеты в коде
- Наличие основных файлов в репозитории

**3 (Удовлетворительно):**
- Базовая адаптивность реализована
- Есть ошибки в отображении на некоторых размерах
- Код требует улучшения
- Не все файлы загружены в репозиторий

**2 (Неудовлетворительно):**
- Адаптивность не работает или работает некорректно
- Серьезные ошибки в коде
- Отсутствие требуемых файлов

### CSS стили
```css
/* Базовые стили (Mobile First) */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

.navbar {
    background: #333;
    padding: 1rem;
    display: flex;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
}

.nav-brand {
    color: white;
    font-size: 1.5rem;
    font-weight: bold;
}

.nav-toggle {
    display: flex;
    flex-direction: column;
    background: none;
    border: none;
    cursor: pointer;
    padding: 0.5rem;
}

.nav-toggle span {
    width: 25px;
    height: 3px;
    background: white;
    margin: 2px 0;
    transition: 0.3s;
}

.nav-menu {
    list-style: none;
    width: 100%;
    display: none;
    flex-direction: column;
}

.nav-menu.active {
    display: flex;
}

.nav-menu li {
    margin: 0.5rem 0;
}

.nav-menu a {
    color: white;
    text-decoration: none;
    padding: 0.5rem;
    display: block;
    transition: background 0.3s;
}

.nav-menu a:hover {
    background: #555;
}

.content {
    padding: 2rem;
}

/* Планшеты */
@media (min-width: 768px) {
    .nav-toggle {
        display: none;
    }
    
    .nav-menu {
        display: flex;
        flex-direction: row;
        width: auto;
        gap: 1rem;
    }
    
    .nav-menu li {
        margin: 0;
    }
}

/* Десктопы */
@media (min-width: 1024px) {
    .navbar {
        padding: 1rem 2rem;
    }
    
    .nav-menu {
        gap: 2rem;
    }
    
    .nav-menu a {
        padding: 0.5rem 1rem;
    }
}
```

### JavaScript для гамбургер-меню
```javascript
document.addEventListener('DOMContentLoaded', function() {
    const navToggle = document.querySelector('.nav-toggle');
    const navMenu = document.querySelector('.nav-menu');
    
    navToggle.addEventListener('click', function() {
        navMenu.classList.toggle('active');
    });
});
```
