# Практическая работа № 4. Работа с коллекциями в Kotlin

## Тема
Использование коллекций (массивы, списки, множества, ассоциативные массивы) в Kotlin.

## Цель
Научиться создавать, модифицировать и использовать различные типы коллекций (Array, List, Set, Map) для решения практических задач. Закрепить навыки работы с циклами, условными выражениями и функциями высшего порядка.

## Теоретическая часть
Коллекции – это структуры данных, которые позволяют хранить группы элементов. В Kotlin основные типы коллекций:
- **Array** – упорядоченная коллекция фиксированного размера.
- **List** – упорядоченная коллекция с возможностью изменения размера.
- **Set** – неупорядоченная коллекция уникальных элементов.
- **Map** – коллекция пар "ключ-значение".

Подробнее с теорией можно ознакомиться в файле `lesson5. Коллекции.md`.

## Практический пример

```kotlin
fun main() {
    // Пример работы со списком
    val planets = mutableListOf("Mercury", "Venus", "Earth", "Mars")
    planets.add("Jupiter")
    println("Third planet: ${planets[2]}")
    
    // Пример работы с ассоциативным массивом
    val moons = mapOf("Earth" to 1, "Mars" to 2)
    println("Moons of Mars: ${moons["Mars"]}")
}
```
