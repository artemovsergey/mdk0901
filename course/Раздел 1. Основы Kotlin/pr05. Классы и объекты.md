## Теоретическая часть

### Классы в Kotlin
Класс — это шаблон для создания объектов. Он определяет свойства (данные) и методы (функции), которые будут у объектов.

**Пример объявления класса:**
```kotlin
class SmartDevice {
    val name = "Android TV"
    var deviceStatus = "online"
    
    fun turnOn() {
        println("Device is turned on")
    }
    
    fun turnOff() {
        println("Device is turned off")
    }
}
```

### Объекты и экземпляры классов
Объект — это экземпляр класса. Мы можем создавать multiple объектов одного класса.

**Пример создания объекта:**
```kotlin
val smartTv = SmartDevice()
smartTv.turnOn()
```

### Конструкторы
Конструкторы используются для инициализации объектов при их создании.

**Пример конструктора:**
```kotlin
class SmartDevice(val name: String, val category: String) {
    var deviceStatus = "online"
    
    // методы класса
}
```

### Наследование
Наследование позволяет создавать новый класс на основе существующего.

**Пример наследования:**
```kotlin
open class SmartDevice(val name: String, val category: String)

class SmartTvDevice(name: String, category: String) : 
    SmartDevice(name, category) {
    // дополнительные свойства и методы
}
```

## Практический пример

**Задача:** Создать систему управления умным домом с устройствами разных типов.

```kotlin
// Базовый класс для всех умных устройств
open class SmartDevice(val name: String, val category: String) {
    var deviceStatus = "online"
        protected set
    
    open fun turnOn() {
        deviceStatus = "on"
        println("$name is turned on")
    }
    
    open fun turnOff() {
        deviceStatus = "off"
        println("$name is turned off")
    }
}

// Класс для умного телевизора
class SmartTvDevice(name: String, category: String) : 
    SmartDevice(name, category) {
    
    private var volume = 5
    private var channel = 1
    
    fun increaseVolume() {
        if (volume < 10) volume++
        println("Volume increased to $volume")
    }
    
    fun changeChannel(newChannel: Int) {
        channel = newChannel
        println("Channel changed to $channel")
    }
    
    override fun turnOn() {
        super.turnOn()
        println("TV is ready with volume $volume and channel $channel")
    }
}

// Класс для умной лампы
class SmartLightDevice(name: String, category: String) : 
    SmartDevice(name, category) {
    
    private var brightness = 50
    
    fun increaseBrightness() {
        if (brightness < 100) brightness += 10
        println("Brightness increased to $brightness")
    }
    
    override fun turnOn() {
        super.turnOn()
        println("Light is on with brightness $brightness")
    }
}

// Класс для управления умным домом
class SmartHome {
    private val devices = mutableListOf<SmartDevice>()
    
    fun addDevice(device: SmartDevice) {
        devices.add(device)
        println("Device ${device.name} added to smart home")
    }
    
    fun turnOnAllDevices() {
        devices.forEach { it.turnOn() }
    }
    
    fun turnOffAllDevices() {
        devices.forEach { it.turnOff() }
    }
    
    fun getDeviceStatus(): String {
        return "Total devices: ${devices.size}. " +
               "Online: ${devices.count { it.deviceStatus == "on" }}"
    }
}

// Использование системы
fun main() {
    val smartHome = SmartHome()
    
    val tv = SmartTvDevice("Living Room TV", "Entertainment")
    val light = SmartLightDevice("Kitchen Light", "Lighting")
    
    smartHome.addDevice(tv)
    smartHome.addDevice(light)
    
    smartHome.turnOnAllDevices()
    tv.increaseVolume()
    tv.changeChannel(5)
    light.increaseBrightness()
    
    println(smartHome.getDeviceStatus())
    
    smartHome.turnOffAllDevices()
}
```
