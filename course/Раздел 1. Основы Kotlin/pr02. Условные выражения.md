## Теоретическая часть

### Условные выражения if/else
Конструкция if/else позволяет выполнять различные блоки кода в зависимости от условия:
```kotlin
if (condition) {
    // код, если условие истинно
} else {
    // код, если условие ложно
}
```

### Множественное ветвление с when
Конструкция when заменяет switch из других языков и позволяет обрабатывать multiple условия:
```kotlin
when (variable) {
    value1 -> действие1
    value2 -> действие2
    else -> действие_по_умолчанию
}
```

### Условные выражения как выражения
В Kotlin if и when могут возвращать значения:
```kotlin
val result = if (x > 0) "Positive" else "Negative"
val message = when (color) {
    "Red" -> "Stop"
    "Green" -> "Go"
    else -> "Unknown"
}
```

## Практический пример

**Задача**: Написать программу, которая определяет время суток по заданному часу.

```kotlin
fun main() {
    val hour = 14
    
    val timeOfDay = when {
        hour < 6 -> "Ночь"
        hour < 12 -> "Утро"
        hour < 18 -> "День"
        hour < 24 -> "Вечер"
        else -> "Неверное время"
    }
    
    println("Сейчас $timeOfDay")
}
```
