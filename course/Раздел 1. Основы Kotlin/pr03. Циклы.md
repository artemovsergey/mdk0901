### Теоретическая часть

Циклы позволяют выполнять блок кода многократно. В Kotlin есть три основных типа циклов:

- **`for`** — используется для перебора элементов коллекции или диапазона.
- **`while`** — выполняется пока условие истинно.
- **`do-while`** — выполняется хотя бы один раз, затем проверяет условие.

#### Примеры:

```kotlin
// Цикл for
for (i in 1..5) {
    println(i) // 1, 2, 3, 4, 5
}

// Цикл while
var x = 5
while (x > 0) {
    println(x)
    x--
}

// Цикл do-while
var y = 0
do {
    println(y)
    y++
} while (y < 3)
```

### Практический пример

```kotlin
fun main() {
    val planets = listOf("Mercury", "Venus", "Earth", "Mars", "Jupiter", "Saturn", "Uranus", "Neptune")

    // Цикл for для перебора списка
    println("Planets in solar system:")
    for (planet in planets) {
        println(planet)
    }

    // Цикл for с индексом
    println("\nPlanets with index:")
    for ((index, planet) in planets.withIndex()) {
        println("Planet ${index + 1}: $planet")
    }

    // Цикл while
    var count = 5
    println("\nCountdown:")
    while (count > 0) {
        println(count)
        count--
    }
}
```
