# Раздел 4. Typescript

- Лекция 11. Введение в TypeScript
  Зачем нужен TS. Установка компилятора, настройка tsconfig.json. Базовые типы: string, number, boolean, array, tuple, enum, any, unknown, void.

- Лекция 12. Система типов и функции
  Собственные типы (type) и интерфейсы (interface). Разница между ними. Типизация функций: типы параметров и возвращаемого значения, опциональные и параметры по умолчанию.

- Лекция 13. ООП в TypeScript
  Классы с модификаторами доступа (public, private, protected). Абстрактные классы. Дженерики (Generics) - создание универсальных функций, классов и интерфейсов.

- Лекция 14. Продвинутые типы и утилиты
  Объединения (Union Types) и пересечения (Intersection Types). Утилиты типов (Partial, Pick, Omit). Приведение типов (Type Assertion). Охранники типов (Type Guards).

- Лекция 15. Интеграция TypeScript в проект
  Типизация сторонних библиотек (@types/...). Работа с DOM в TS. Использование TS с современными инструментами сборки (Vite, Webpack).

- Практическая работа 12. Базовые типы и интерфейсы
  Типизация данных из Практической работы 7. Создание интерфейсов для объектов пользователя, товара и т.д.

- Практическая работа 13. Типизация функций и классов
  Переписывание функций и классов из раздела JS с добавлением строгой типизации. Использование дженериков для создания функции-обертки для API-запроса.

- Практическая работа 14. Утилиты типов и охранники
  Создание гибких типов на основе базовых с помощью Partial и Pick. Написание функций-охранников для сужения типов.

- Практическая работа 15. Миграция To-Do List на TypeScript
  Поэтапный перенос мини-проекта из JS в TS: типизация всех переменных, функций, классов и данных.

- Практическая работа 16. Типизация асинхронных операций
  Создание типизированной функции-клиента для работы с HTTP API. Типизация ответов и ошибок.

- Самостоятельная работа 5. Рефакторинг кода и поиск ошибок
  Дан фрагмент кода на JS с типичными ошибками. Задача: переписать его на TypeScript, исправив ошибки на этапе компиляции и добавив недостающие типы.
